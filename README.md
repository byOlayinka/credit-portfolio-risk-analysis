# Credit Portfolio Risk Analysis Using Anonymised Loan Application Data

## Project overview

This repository contains the public support files for my MBA Data Analytics 2 case study:

**Credit Portfolio Risk Analysis Using Anonymised Loan Application Data**

The analysis uses anonymised loan application and repayment data from Renmoney Microfinance Bank to examine repayment behaviour, default risk, credit score patterns, and repayment probability.

## Live report

RPubs report:

https://rpubs.com/yinkayinkayinka/Credit_Portfolio_Risk_Analysis_Using_Anonymised_Loan_Application_Data

## Important data confidentiality note

The original dataset used for the analysis is confidential internal organisational data. It is not included in this public repository because it contains internal business information, even though personally identifiable customer information was removed before analysis.

This repository therefore includes:

- The Quarto source file for the analysis, to be uploaded separately by the author.
- A data dictionary showing the variables used in the analysis.
- A sample data structure file with dummy values only.
- A link to the published RPubs report.

The sample data structure file is not the original dataset and should not be used to reproduce the report outputs. It is included only to show the structure of the fields used in the analysis.

## Analysis techniques used

The report applies five required techniques:

1. Exploratory Data Analysis
2. Data Visualisation
3. Hypothesis Testing
4. Correlation Analysis
5. Logistic Regression

## Reproducibility note

The published RPubs report contains the full analysis code, outputs, visualisations, and interpretations. The raw dataset is withheld due to confidentiality restrictions, so this public repository is designed for transparency around the source code and data structure, not public release of internal data.
